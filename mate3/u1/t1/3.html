Razones trigonométricas para ángulos agudos de un triángulo rectángulo

Figura 1 “a y b son los catetos(lados que forman el ángulo recto) y c la hipotenusa ( lado contrario al ángulo recto)”.

Como puedes observar “a” es el cateto opuesto al ángulo $\alpha$ , ”b” es el cateto adyacente al ángulo $\alpha$.

Para el ángulo $\beta$ “a” es el cateto adyacente y “b” es el cateto opuesto.

Y tanto para el ángulo $\alpha$ como para ángulo $\beta$ “c” es la hipotenusa.

La razón o cociente $\frac{a}{c}$ de se le denomina “seno” del ángulo “$\alpha$”. Las razones trigonométricas son seis: seno, coseno y tangente; y las recíprocas de estas son cosecante, secante y cotangente respectivamente.

\[sen\left( \alpha \right) = \frac{{{{cateto \hspace{1 cm}opuesto}}}}{{{{hipotenusa}}}}=\frac{{{{a}}}}{{{{c}}}}\]

\[cos\left( \alpha \right) = \frac{{{{cateto \hspace{1 cm} adyacente}}}}{{{{hipotenusa}}}}=\frac{{{{b}}}}{{{{c}}}}\]

\[tan\left( \alpha \right) = \frac{{{{cateto \hspace{1 cm} opuesto}}}}{{{{cateto \hspace{1 cm} adyacente}}}}=\frac{{{{a}}}}{{{{b}}}}\]

\[csc\left( \alpha \right) = \frac{{{{hipotenusa}}}}{{{{cateto \hspace{1 cm} opuesto}}}}=\frac{{{{c}}}}{{{{a}}}}\]

\[sec\left( \alpha \right) = \frac{{{{hipotenusa}}}}{{{{cateto \hspace{1 cm} adyacente}}}}=\frac{{{{c}}}}{{{{b}}}}\]

\[cot\left( \alpha \right) = \frac{{{{cateto \hspace{1 cm} adyacente}}}}{{{{cateto \hspace{1 cm} opuesto}}}}=\frac{{{{b}}}}{{{{a}}}}\]
