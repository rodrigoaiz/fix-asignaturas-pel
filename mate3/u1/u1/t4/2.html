Ejemplos de Aplicaciones con Identidades Trigonométricas.

Sabiendo que $tan \alpha = \sqrt{3}$ y $ 0^{\circ} < \alpha < 90^{\circ}$ , calcula las razón trigonométrica de secante $\alpha$

\[1 +tan^2 { {\alpha}} = sec^2{ {\alpha}}\]

\[1 +(\sqrt{3})^{2} = sec^2{ {\alpha}}\]

\[1 +3 =sec^2{ {\alpha}}\]

\[4=sec^2{ {\alpha}}\]

\[sec{ {\alpha}}= \pm\sqrt{2}\]

Sabiendo que sen $ \alpha = \frac{3}{5}$ y $ 0^{\circ} < \alpha < 90^{\circ}$ , calcula las razón trigonométrica de coseno $\alpha$

Utilizando la siguiente identidad trigonométrica.

\[sen^2{ {\alpha}}+cos^2 { {\alpha}} = 1\]

Sustituyendo los valores y despejando coseno

\[cos^2 { {\alpha}} = 1-\frac{{{{3^2}}}}{{{{5^2}}}}\]

\[cos^2 { {\alpha}} = 1-\frac{{{{9}}}}{{{{25}}}}\]

\[cos^2 { {\alpha}} = \frac{{{{16}}}}{{{{25}}}}\]

\[cos { {\alpha}} = \pm\sqrt\frac{{{{16}}}}{{{{25}}}}\]

\[cos { {\alpha}} = \pm \frac{{{{4}}}}{{{{5}}}}\]
