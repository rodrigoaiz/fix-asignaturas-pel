Actividad 1

Para saber si dos rectas son paralelas o perpendiculares debemos de identificar las condiciones que se presentan para cada caso.

En esta actividad realizaremos ejemplos que puedan ayudarte para identificar si 2 rectas son Paralelas (||) ó Perpendiculares (⊥).

Ejemplo 1

Determina si la recta que pasa por los puntos A(6,0) ; B(0,4) y la recta que pasa por los puntos C(0,2) ; D(3,0) son Paralelas.

p.font-bold Solución:

Se debe determinar la pendiente (m) de cada una de las rectas con los puntos dados de cada recta. Aplicando la fórmula de la pendiente esto es:

Para los Puntos A y B se tiene que $x_1 = 6$ ; $x_2 = 0$ ; $y_1 = 0$ ; $y_2=0$

$m_{AB} = \frac{y_2 - y_1}{x_2 - x_1} = \frac{4-0}{0-6} = \frac{4}{-6} = - \frac{2}{3}$

$m_{AB} = - \frac{2}{3}$

Para los Puntos C y D se tiene que: $x_1 = 0$ ; $x_2 = 3$ ; $y_1 = 2$ ; $y_2=0$

$m_{CD} = \frac{y_2 - y_1}{x_2 - x_1} = \frac{0-2}{3-0} = \frac{-2}{3} = - \frac{2}{3}$

$m_{CD} = - \frac{2}{3}$

Por lo tanto como $m_{AB} = m_{CD}$ Esto implica que las rectas son paralelas.

Ejemplo 2

Demuestra que la recta que pasa por los puntos A(2,5) ; B(-3-2) es perpendicular a la recta que pasa por los puntos C(4,-1) ; $D(-\frac{8}{5} , 3)$.

Nuevamente lo primero que se debe de realizar es encontrar la pendiente de cada recta, Utilizando la fórmula:

$m = \frac{y_2 - y_1}{x_2 - x_1}$

Para los puntos A y B se tiene que: $x_1 = 2$ ; $x_2 = -3$ ; $y_1 = 5$ ; $y_2=-2$

$m_{AB} = \frac{y_2 - y_1}{x_2 - x_1} = \frac{-2-5}{-3-2} = \frac{-7}{-5} = \frac{7}{5}$

$m_{AB} = \frac{7}{5} $

Para los puntos C y D se tiene que: $x_1 = 4$ ; $x_2 = -\frac{8}{5}$ ; $y_1 = -1$ ; $y_2=3$

$m_{CD} = \frac{y_2 - y_1}{x_2 - x_1} = \frac{3-(-1)}{-\frac{8}{5}-4} = \frac{3+1}{-\frac{28}{5}} = \frac{4}{-\frac{28}{5}} = -\frac{5}{7}$

$m_{CD} = - \frac{5}{7}$

Por lo tanto como $m_{AB} * m_{CD}$ Esto implica que producto de sus pendientes debe cumplir que: $m_1 * m_2 = -1$ por lo tanto:

$m_{AB} * m_{CD} = -1$

$\frac{7}{5} * -\frac{5}{7} = 1$

Las rectas son perpendiculares porque cumplen la condición de perpendicularidad.
